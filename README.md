# allMyCats1.py

**Program Description**

This Python script is designed to collect and display the names of six cats from user input.

**Functionality**

1. The program prompts the user to enter the name of six cats, one at a time, using the `input()` function.
2. The user's input is stored in six separate variables: `catName1`, `catName2`, ..., `catName6`.
3. After collecting all six names, the program displays a message indicating that the cat names are being displayed.
4. Finally, the program prints the names of all six cats, separated by spaces, using the `print()` function.
